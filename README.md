# Circuito Elegante - AI Agent Project

AI-powered hotel concierge agent for the Circuito Elegante hotel network, integrated with Elevare API.

## 🎯 Project Overview

- **Scope:** AI agent for customer service & hotel operations across Circuito Elegante network
- **Integration:** Elevare API for hotel management system
- **Framework:** Synkra AIOX (AI-Orchestrated Development)
- **Status:** Initialization phase

## 📁 Project Structure

```
.
├── docs/              # Documentation (stories, PRDs, architecture)
├── squads/            # Squad-specific implementations
├── presentation/      # Partnership presentation materials
├── .aiox-core/        # AIOX framework (framework-protected)
└── .claude/           # Claude Code configuration
```

## 🚀 Quick Start

1. **Review the PRD:** See `docs/prd/` for feature specifications
2. **Check stories:** Development stories in `docs/stories/`
3. **Activate AIOX:** Type `@aiox-master` to begin orchestration

## 📋 Development Workflow

This project uses **Story-Driven Development**:
- All work originates from stories in `docs/stories/`
- Stories track acceptance criteria and progress
- Development follows AIOX agent delegation patterns

## 🔗 Key Integration

- **Elevare API:** Hotel management system integration
- **Hotel Network:** Multi-property support across Circuito Elegante
