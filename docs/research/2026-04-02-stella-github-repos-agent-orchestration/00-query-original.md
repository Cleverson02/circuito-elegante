# Query Original

## Pergunta

> Baseado no PRD da Stella (Concierge Digital do Circuito Elegante), existem repositorios no GitHub ou outro local que nos atendem com sistema ja consolidado e funcionando, com estrutura semelhante que faz a orquestracao de agentes? Apenas repositorios com 5+ estrelas, primando por qualidade, para identificar o que pode nos ajudar a acelerar o processo.

## Contexto Inferido

- **Foco:** Tecnico (GitHub repos, agent orchestration, WhatsApp)
- **Dominio:** OpenAI, WhatsApp, Multi-Agent, TypeScript, Fastify, Redis, Supabase
- **Temporal:** 2025-2026 (estado da arte)
- **PRD Base:** Stella Concierge Digital — 4 subagentes (Intent -> Orchestrator -> Persona -> Safety), Evolution API, Chatwoot, Fastify, BullMQ, pgvector, RAG

## Criterios de Busca

1. Repositorios com 5+ estrelas no GitHub
2. Sistemas consolidados e funcionais
3. Arquitetura de orquestracao multi-agente
4. Compatibilidade com stack do PRD (TypeScript, OpenAI, WhatsApp)
5. Qualidade de codigo e documentacao
