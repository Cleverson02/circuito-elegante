## Descrição

Criar a tool `searchHotels` que busca hotéis no PostgreSQL com filtros dinâmicos via Drizzle ORM. É a principal tool do Orchestrator Agent para responder buscas de hotéis por critérios como região, experiência, destino, pet-friendly, etc.

**Problema/Necessidade:** O Orchestrator precisa buscar hotéis no banco de dados baseado nos critérios extraídos da intenção do usuário. Sem esta tool, não é possível responder queries de busca.
