## Descrição

Criar o Orchestrator Agent usando GPT-5-turbo (ou GPT-4o) via OpenAI Agents SDK. O Orchestrator é o segundo estágio do pipeline — recebe a intenção classificada e executa as tools apropriadas (searchHotels, queryKnowledgeBase, transferToHuman). Faz handoff para o Persona Agent após obter resultados.

**Problema/Necessidade:** O pipeline precisa de um agente que mapeie intenções para tools e execute-as. Sem o Orchestrator, não há como conectar Intent → Tools → Persona.
