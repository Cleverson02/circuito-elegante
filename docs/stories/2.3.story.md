## Descrição

Criar a tool `queryKnowledgeBase` que busca informações de FAQ via RAG usando pgvector. Recebe uma pergunta, gera embedding, busca chunks similares no banco e retorna os top 3 resultados. Inclui fuzzy match de nome de hotel e cache de embeddings frequentes via Redis.

**Problema/Necessidade:** O sistema RAG da Stella depende desta tool para responder perguntas sobre hotéis, amenities e políticas usando os FAQ embeddings ingeridos na Story 1.4.
