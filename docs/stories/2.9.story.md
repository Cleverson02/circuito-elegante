## Descrição

Criar o Safety & Validation Agent usando GPT-5-nano que audita a resposta do Persona Agent antes de enviá-la ao usuário. Verifica persona consistency, dados inventados, tom adequado, segurança e idioma correto. É o último estágio do pipeline — gate final antes da entrega.

**Problema/Necessidade:** O Persona Agent pode alucinar dados, quebrar persona, ou gerar conteúdo inadequado. O Safety Agent é a última linha de defesa antes da resposta chegar ao guest.

## Acceptance Criteria

- [ ] AC1: Safety Agent com GPT-5-nano (~50ms) via OpenAI Agents SDK
- [ ] AC2: Checklist de 5 pontos: persona, dados, tom, segurança, idioma

- [ ] AC6: Métricas: taxa rejeição por categoria
- [ ] AC7: Testes: IA mencionada→REJECTED, preço inventado→REJECTED, escassez artificial→REJECTED
