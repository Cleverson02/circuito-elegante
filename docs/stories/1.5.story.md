## Descrição

Configurar Redis 7 Alpine via Docker Compose com volume persistente, criar client com retry logic, implementar helpers de session/request management, e middleware de rate limiting. Esta story estabelece a fundação de estado transiente para o pipeline de agentes da Stella.

**Problema/Necessidade:** O pipeline multi-agente precisa de session state (conversa ativa, requestId, tokens opaco) e rate limiting (30 msgs/min). Redis é a escolha arquitetural para dados transientes com TTL.
