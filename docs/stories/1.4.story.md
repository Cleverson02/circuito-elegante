## Descrição

Criar pipeline de ingestão de FAQs do Google Drive para pgvector. O script conecta à Google Drive API v3, baixa arquivos de FAQ (.md, .txt, .docx), aplica semantic chunking por seção lógica, gera embeddings via `text-embedding-3-small` (1536 dimensões) e armazena na tabela `faq_embeddings` com deduplicação por `content_hash` (SHA-256).

**Problema/Necessidade:** O sistema RAG da Stella depende dos FAQ embeddings para responder perguntas sobre hotéis, amenities, políticas e procedimentos. Sem embeddings, o agente não tem base de conhecimento.
