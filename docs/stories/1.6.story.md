## Descrição

Consolidar o health check para retornar status de todas as dependências (database, redis, vectordb, hotels_count) e criar smoke test suite que valida o funcionamento end-to-end de todo o EPIC-1. Atualizar CI para incluir smoke tests.

**Problema/Necessidade:** Após todas as stories de fundação, precisamos de uma validação end-to-end que confirme que todos os componentes estão funcionando em conjunto. É o quality gate final do EPIC-1.

- [ ] AC1: `GET /health` retorna status consolidado (database, redis, vectordb, hotels_count)
