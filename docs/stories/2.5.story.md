## Descrição

Criar o Persona Agent usando GPT-5-pro (ou GPT-4o) que transforma dados brutos dos tools em prosa AAA (Acolhedora, Conhecedora, Discreta) na voz da Stella. O Persona é o terceiro estágio do pipeline — recebe resultados de tools e gera resposta natural no idioma do guest.

**Problema/Necessidade:** Os tools retornam dados brutos (JSON). O Persona Agent transforma em resposta humana, mantendo tom de concierge de luxo, sem expor IDs técnicos, e curadoriando 3-4 opções quando há múltiplos resultados.

## Dev Notes

- **Pipeline:** Persona recebe dados do Orchestrator, NUNCA chama tools
- **Tom AAA:** Acolhedora (warm), Conhecedora (knowledgeable), Discreta (discreet)
- **Preços:** Sempre em BRL (R$), formatados
- **Opções:** Quando >4 resultados, curadoriar top 3-4 com destaque
- **Idioma:** Responder no idioma detectado pelo Intent Agent
- Referência: `docs/architecture/architecture.md` seção 5.3
