## Descrição

Criar a estrutura base do monorepo para o projeto Stella (Concierge Digital do Circuito Elegante). Esta story estabelece o scaffold completo: servidor Fastify com health check, Docker Compose (app + Redis), configuração de ambiente com validação Zod, linting, testes, CI, logging e monitoring. É o pré-requisito para todas as demais stories do EPIC-1.

**Problema/Necessidade:** Sem scaffold, nenhuma outra story pode ser implementada. O projeto precisa de uma base sólida com boas práticas desde o início.

**Referências:**
- Estrutura: `docs/architecture/architecture.md` seção 2.2
- Env vars: `docs/architecture/architecture.md` seção 15
- PRD: `docs/prd/PRD-Concierge-Digital-Stella.md`
