# Secret Formula Checklist — Kennedy's Message-Market-Media

## Source
Dan Kennedy, "No B.S. Direct Marketing" + "Ultimate Marketing Plan"

## Purpose
Validate the core marketing formula: Results = RIGHT Message + RIGHT Market
+ RIGHT Media. This is the foundation of ALL Kennedy marketing. Nothing
works without this formula being correct.

Kennedy calls this the "Results Triangle" — and it must be validated
before any campaign, letter, ad, or strategy is designed.

---

## THE FORMULA

```
RESULTS = RIGHT Message + RIGHT Market + RIGHT Media
```

Each element must be independently strong AND aligned with the other two.
A brilliant message to the wrong market = zero results.
The right market with the wrong media = wasted money.

---

## SECTION 1: RIGHT MARKET (Who)

### Market Identification
- [ ] Target market defined with specificity (not "small business owners")
- [ ] Demographics documented:
  - [ ] Age range
  - [ ] Income level / business revenue
  - [ ] Geographic location
  - [ ] Industry/occupation
  - [ ] Education level
  - [ ] Gender (if relevant)

### Psychographic Profile
- [ ] Fears documented (what are they afraid of?)
- [ ] Frustrations documented (what makes them angry?)
- [ ] Desires documented (what do they desperately want?)
- [ ] Aspirations documented (who do they want to become?)
- [ ] Current beliefs about the problem documented
- [ ] Current beliefs about solutions documented

### Money Pyramid Position
- [ ] Target tier identified:
  - [ ] 1% Ultra-rich (price is irrelevant, time is everything)
  - [ ] 4% Rich (value-driven, expect premium experience)
  - [ ] 15% Comfortable (quality-conscious, comparison shoppers)
  - [ ] 60% Paycheck-to-paycheck (price-sensitive, payment plans)
  - [ ] 20% Broke (avoid — highest service cost, lowest lifetime value)
- [ ] Pricing aligned with target tier
- [ ] Language aligned with target tier
- [ ] Service delivery aligned with target tier

### Starving Crowd Test
- [ ] Is this market HUNGRY for a solution? (not just interested)
- [ ] Is this market REACHABLE through identifiable channels?
- [ ] Is this market ABLE TO PAY for the solution?
- [ ] Is this market LARGE ENOUGH to sustain the business?
- [ ] Does this market have EXISTING buying behavior for similar solutions?

### Congregation Mapping
- [ ] Where does this market congregate ONLINE?
  - [ ] Social media groups / pages
  - [ ] Forums / communities
  - [ ] YouTube channels
  - [ ] Podcasts
  - [ ] Newsletters / blogs
- [ ] Where does this market congregate OFFLINE?
  - [ ] Industry events / conferences
  - [ ] Associations / clubs
  - [ ] Local businesses they visit
  - [ ] Publications they read
- [ ] Minimum 10 congregation sources identified

## SECTION 2: RIGHT MESSAGE (What)

### Big Promise
- [ ] ONE primary promise articulated clearly
- [ ] Promise addresses the market's #1 desire or #1 pain
- [ ] Promise is specific (includes numbers, timeframes, or outcomes)
- [ ] Promise is believable (not too good to be true)
- [ ] Promise is unique to you (not what everyone else says)

### USP (Unique Selling Proposition)
- [ ] USP answers: "Why you vs. every other option, including doing nothing?"
- [ ] USP is stated in ONE clear sentence
- [ ] USP uses one of Kennedy's 4 Pathways:
  - [ ] Be the FIRST (pioneer positioning)
  - [ ] Be the BEST (superiority with proof)
  - [ ] Be DIFFERENT (contrarian/unique approach)
  - [ ] Be SPECIALIZED (niche domination)
- [ ] USP is provable (not just a claim)
- [ ] USP is used consistently across all materials

### Message Architecture
- [ ] Problem is clearly stated (in the prospect's words)
- [ ] Problem is agitated (made more urgent and painful)
- [ ] Solution is presented as the logical answer
- [ ] Proof supports the claims (testimonials, case studies, data)
- [ ] Transformation is described (before/after state)

### Message Tone
- [ ] Message sounds like one person talking to another
- [ ] Message uses the prospect's language (not jargon)
- [ ] Message is direct and assertive (not meek or corporate)
- [ ] Message is honest (includes "damaging admission" where appropriate)
- [ ] Message would pass Kennedy's "read aloud" test

## SECTION 3: RIGHT MEDIA (How)

### Media Selection
- [ ] Media channels selected based on WHERE the market congregates
- [ ] Media matches the complexity of the message
- [ ] Media budget justified by expected ROI
- [ ] Multiple media channels in use (not dependent on one)

- [ ] Testing budget allocated for new channels

## SECTION 4: ALIGNMENT CHECK

### Message-Market Alignment
- [ ] Message addresses what the market actually cares about
- [ ] Message uses language the market uses (not your jargon)
- [ ] Message tone matches market sophistication level
- [ ] Message price point matches market's Money Pyramid tier
- [ ] Market would recognize themselves in the message

### Market-Media Alignment
- [ ] Media channels reach where the market actually IS
- [ ] Media format matches market's consumption habits
- [ ] Media timing matches when the market is receptive
- [ ] Media investment is proportional to market value

### Message-Media Alignment
- [ ] Message complexity matches media format
- [ ] Long message = long-form media (letters, video, webinars)
- [ ] Simple message = short-form media (postcards, ads)
- [ ] Media allows enough space for the complete message
- [ ] Media supports the proof elements needed

### Full Triangle Alignment
- [ ] All three elements work together harmoniously
- [ ] No single element undermines the other two
- [ ] If any element were changed, the others would need to change too
- [ ] The triangle would survive Kennedy's "brutal honesty" review
