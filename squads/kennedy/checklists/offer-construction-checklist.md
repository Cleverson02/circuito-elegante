# Irresistible Offer Construction Checklist

> Agent: kennedy-offers
> Source: The Ultimate Marketing Plan + Brass Balls Course — Offer Architecture
> Version: 1.0
> Use: When building an offer stack that makes it hard to say no

---

## Pre-Requisites
- [ ] Product or service clearly defined
- [ ] Target market identified and understood
- [ ] Price point determined
- [ ] Competitor offers researched

---

## STEP 1: The Core Offer

### 1.1 Value Proposition
- [ ] Core offer delivers a clear, measurable result
- [ ] The transformation is stated in customer language
- [ ] The benefit is specific (not "improve your life")
- [ ] The core offer alone justifies the price

### 1.2 Widget Concept
- [ ] Tangible "widget" component included (something physical to hold)
- [ ] Widget has high perceived value but low cost to produce
- [ ] Widget differentiates the offer from pure information/service
- [ ] Widget is "keep regardless" (even if they return main offer)
- [ ] Examples: toolkit, physical book, checklist card, calculator, kit

### 1.3 Three Types of Offers Applied
- [ ] Type identified: Free (lead gen) / Low-barrier (trial) / Full price
- [ ] Offer type matches the stage in the customer journey
- [ ] Each type has appropriate risk reversal
- [ ] Ascension path from lower to higher offer types planned

---

## STEP 2: Bonus Stack

### 2.1 Bonus Design
- [ ] Minimum 3 bonuses included
- [ ] Each bonus has a specific name and stated value
- [ ] Bonus #1 solves an adjacent problem
- [ ] Bonus #2 accelerates results or saves time
- [ ] Bonus #3 eliminates the most common excuse for not acting

### 2.2 Bonus Value
- [ ] Each bonus has a believable standalone value
- [ ] Total bonus value exceeds the price of the main offer
- [ ] At least one time-sensitive bonus (fast-action incentive)
- [ ] Bonuses feel genuinely valuable (not filler)

---

## STEP 3: Guarantee Design

### 3.1 Risk Reversal
- [ ] Guarantee clearly stated (duration, terms, process)
- [ ] Guarantee is prominently displayed (not buried in fine print)
- [ ] Guarantee language is confident (not apologetic)
- [ ] Guarantee shifts ALL risk from buyer to seller

### 3.2 Guarantee Level
- [ ] Standard: Money-back (minimum baseline)
- [ ] Better: Money-back PLUS keep bonuses
- [ ] Best: Double-your-money-back or performance guarantee
- [ ] Kennedy: "If you truly believe in your product, offer the strongest possible guarantee"

---

## STEP 4: Urgency and Scarcity

### 4.1 Deadline
- [ ] Specific deadline stated (date or countdown)
- [ ] Consequence of missing deadline clear (price increase, bonuses removed)
- [ ] Deadline is REAL and enforced (credibility depends on it)
- [ ] Reason for deadline explained (not arbitrary)

### 4.2 Scarcity
- [ ] Limited quantity or availability stated if applicable
- [ ] Scarcity is genuine and justified
- [ ] Evidence of scarcity provided (previous sold-out events, waitlists)
- [ ] Creates "fear of missing out" without manipulation

---

## STEP 5: Offer Presentation

### 5.1 Value Anchoring
- [ ] Individual values stated for each component
- [ ] Total value calculated and displayed prominently
- [ ] Price revealed AFTER value stack is built
- [ ] Price feels small compared to total value (minimum 5:1 ratio)

### 5.2 Comparison Frame
- [ ] Offer compared to more expensive alternatives
- [ ] "If you hired me personally, it would cost $___, but..."
- [ ] Daily cost breakdown ("less than a cup of coffee per day")
- [ ] Investment frame used (not "cost" or "expense")

---

## Validation
- [ ] Offer passes the "would I buy this?" test
- [ ] A skeptical friend finds it compelling when explained
- [ ] Value math checks out (total value > 5x price)
- [ ] Every common objection has an answer within the offer
- [ ] Offer can be clearly explained in 60 seconds

---

## Scoring
- 34-38/38 = Irresistible offer — deploy immediately
- 28-33/38 = Strong offer — strengthen weak elements
- 20-27/38 = Good foundation — missing key components
- Below 20 = Offer needs significant redesign
