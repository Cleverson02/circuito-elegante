# Funnel Audit Checklist — Kennedy-Style DR Campaign Review

## Source
Dan Kennedy, compiled from: No B.S. Direct Marketing, Ultimate Marketing Plan,
Magnetic Marketing, Midas Touch Course

## Purpose
Audit a marketing campaign or "funnel" (Kennedy uses "campaign" not "funnel")
through the lens of Kennedy's direct response principles. This is NOT about
page sequences or click-through rates — it is about whether the CAMPAIGN as
a whole follows proven DR principles that produce measurable results.

**Note:** Kennedy does not use "funnel" language. He builds CAMPAIGNS with
multiple touchpoints. This checklist evaluates the complete campaign system.

---

## SECTION 1: Campaign Foundation

### Results Triangle
- [ ] Message clearly defined and compelling
- [ ] Market specifically targeted (starving crowd identified)
- [ ] Media selected and appropriate for the market
- [ ] All three aligned and reinforcing each other

### USP Integration
- [ ] USP is stated clearly in the campaign
- [ ] USP differentiates from all competitors
- [ ] USP is provable (not just a claim)
- [ ] USP runs through every touchpoint consistently

### Offer Architecture
- [ ] Primary offer is clear and compelling
- [ ] Offer includes risk reversal (guarantee)
- [ ] Offer has urgency mechanism (real deadline)
- [ ] Offer has value stacking (perceived value >> price)
- [ ] Offer includes premium/widget where appropriate

---

## SECTION 2: Lead Generation (Entry Point)

### Lead Capture
- [ ] Lead magnet has high perceived value
- [ ] Lead magnet solves a specific problem
- [ ] Opt-in mechanism is simple (minimum friction)
- [ ] Lead capture exists on EVERY touchpoint (not just one page)
- [ ] Lead capture collects enough data for follow-up (name, email, phone, address)

### A-Pile / B-Pile Test (for mail-based campaigns)
- [ ] Mail piece would land in the A-Pile (personal, not bulk)
- [ ] Envelope design demands opening
- [ ] First impression triggers curiosity, not deletion/trash
- [ ] Grabber/lumpy mail element used if targeting high-value prospects

### Ad Copy (for ad-based entry)
- [ ] Headline stops the scroll / captures attention
- [ ] Ad is DR-compliant (offer, CTA, tracking)
- [ ] Ad targets specific audience (not "everyone")
- [ ] Multiple ad variations being tested

---

## SECTION 3: Conversion Sequence

### Multi-Step Process
- [ ] Campaign uses multiple touchpoints (not single-touch)
- [ ] Minimum 5-step follow-up sequence designed
- [ ] Each step has a different angle or approach
- [ ] Urgency escalates through the sequence
- [ ] Multiple media used (mail, email, phone, video)

### Shock-and-Awe Integration
- [ ] Shock-and-awe package deployed to qualified leads
- [ ] Package has minimum 5 high-value elements
- [ ] Package builds authority and trust
- [ ] Package includes a time-sensitive offer
- [ ] Package delivery is tracked

### Persuasion Elements
- [ ] Social proof throughout (testimonials, case studies)
- [ ] Authority established (credentials, media, endorsements)
- [ ] Scarcity is real and communicated
- [ ] Objections addressed pre-emptively
- [ ] Fascination bullets used in key pieces
- [ ] Emotional triggers layered through the sequence

### Conversion Point
- [ ] Clear, specific call to action
- [ ] Multiple response channels (phone, web, mail)
- [ ] Response mechanism is EASY (not complicated)
- [ ] Order process is simple and fast
- [ ] Confirmation and next steps communicated immediately

---

## SECTION 4: Back-End and Ascension

### Immediate Back-End
- [ ] Thank-you page/letter includes additional offer
- [ ] Upsell or cross-sell presented at point of purchase
- [ ] New customer onboarding sequence designed
- [ ] "Buyer's remorse" prevention communication sent

### Customer Ascension
- [ ] Higher-tier offers available for proven buyers
- [ ] Ascension path is natural (value increases with price)
- [ ] VIP/premium program available
- [ ] Repeat purchase incentives in place

### Lifetime Value Optimization
- [ ] Customer LTV calculated and tracked
- [ ] Back-end revenue exceeds front-end acquisition cost
- [ ] Monthly/quarterly communication maintains relationship
- [ ] Customer purchase frequency is measured and optimized

---

## SECTION 5: Tracking and Measurement

### Response Tracking
- [ ] Unique tracking code on every piece
- [ ] Unique phone number per campaign (or extension)
- [ ] Unique URL or landing page per campaign
- [ ] Source attribution on every new lead/customer
- [ ] Response rate tracked by step in the sequence

### Financial Tracking
- [ ] Cost per lead calculated
- [ ] Cost per acquisition calculated
- [ ] Campaign ROI calculated (revenue / spend)
- [ ] Lifetime value factored into ROI
- [ ] Break-even point identified

### Optimization
- [ ] A/B testing planned for key elements
- [ ] Test results documented and acted upon
- [ ] Winning variations scaled, losers killed
- [ ] Testing budget allocated (10-20% of campaign budget)

---

## SECTION 6: Referral Integration

### Built-In Referral Mechanisms
- [ ] Referral request integrated at peak satisfaction point
- [ ] Referral tools provided (cards, scripts, links)
- [ ] Referral incentive offered
- [ ] Referred leads enter campaign with special path
- [ ] Referral results tracked and measured

## Campaign Health Score

| Section | Max Points | Score |
|---------|-----------|-------|
| Campaign Foundation | 15 | /15 |
| Lead Generation | 15 | /15 |
| Conversion Sequence | 30 | /30 |
| Back-End + Ascension | 20 | /20 |
| Tracking + Measurement | 15 | /15 |
| Referral Integration | 5 | /5 |
| **TOTAL** | **100** | **/100** |

| Score | Rating | Action |
|-------|--------|--------|
| 85-100 | Strong Campaign | Optimize and scale |
| 70-84 | Solid Foundation | Fix gaps, then scale |
| 50-69 | Significant Gaps | Major work before scaling |
| Below 50 | Campaign Crisis | Rebuild using Kennedy principles |

---

## Kennedy's Final Test

Ask yourself:
- [ ] "Would I spend my OWN money on this campaign?"
- [ ] "Is every dollar tracked and accountable?"
- [ ] "Could I explain to Dan Kennedy why each piece exists?"
- [ ] "If I removed any element, would results drop?"

If all YES: your campaign is Kennedy-approved.
If any NO: fix that element before spending another dollar.
