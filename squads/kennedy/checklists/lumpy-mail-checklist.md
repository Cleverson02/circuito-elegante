# Lumpy / Grabber Mail Checklist

> Agent: kennedy-media
> Source: No B.S. Direct Marketing + Ultimate Sales Letter — Dimensional Mail Strategy
> Version: 1.0
> Use: When creating dimensional or grabber mail pieces

---

## Pre-Requisites
- [ ] Target recipient identified (high-value prospect or customer)
- [ ] Budget per piece allows for dimensional elements
- [ ] Message and offer defined
- [ ] Mailing quantity determined

---

## STEP 1: Grabber Selection

### 1.1 Physical Object
- [ ] Object relates to the message (not random novelty)
- [ ] Object creates curiosity when felt through the envelope/package
- [ ] Object has a metaphorical connection to the offer
- [ ] Object is memorable and conversation-worthy

### 1.2 Common Grabber Options
- [ ] Dollar bill (or play money) — "Don't throw this away with a dollar attached"
- [ ] Key — "The key to [desired outcome]"
- [ ] Compass — "Finding your direction to [result]"
- [ ] Seed packet — "Plant the seeds of [success]"
- [ ] Aspirin — "Relief from your [headache/problem]"
- [ ] Rubber band — "Are you stretched too thin?"
- [ ] Small toy — relevant to metaphor in the letter
- [ ] Custom item — unique to your industry

### 1.3 Object-Message Connection
- [ ] First paragraph of letter references the object
- [ ] Object becomes a metaphor for the core message
- [ ] Connection is clever but not confusing
- [ ] Object makes the letter memorable

---

## STEP 2: Package Design

### 2.1 Format
- [ ] Package cannot be easily mistaken for standard mail
- [ ] Weight and shape create curiosity
- [ ] Unboxing experience designed (not just an envelope)
- [ ] Package survives shipping without damage to grabber

### 2.2 A-Pile Enhancement
- [ ] All standard A-Pile criteria met
- [ ] Plus: package is physically impossible to ignore
- [ ] Recipient will show it to others (conversation starter)
- [ ] Package sits on desk as a reminder (not thrown away)

---

## STEP 3: Letter Integration

### 3.1 Opening
- [ ] "You're probably wondering why I sent you [object]..."
- [ ] Immediate connection between object and prospect's problem
- [ ] Transition from curiosity about object to interest in offer
- [ ] Object reference maintained throughout (not one-time mention)

### 3.2 Complete DR Elements
- [ ] Full sales letter included (not just a note)
- [ ] Offer, deadline, CTA all present
- [ ] Tracking mechanism (unique code or phone number)
- [ ] P.S. lines reference the object and the offer

---

## STEP 4: Economics

### 4.1 Cost-Benefit
- [ ] Cost per piece calculated (including object, packaging, postage)
- [ ] Break-even response rate calculated
- [ ] High enough value proposition to justify the investment
- [ ] ROI tracked against standard mail benchmark

### 4.2 Targeting
- [ ] Lumpy mail sent to highest-value prospects (not mass market)
- [ ] List quality justifies the per-piece investment
- [ ] Segmentation ensures relevance
- [ ] Volume appropriate for budget and target

---

## Validation
- [ ] Grabber connects meaningfully to the message
- [ ] Package creates physical curiosity and excitement
- [ ] Full DR letter with offer, deadline, CTA included
- [ ] Cost per piece within budget
- [ ] Tracking in place for ROI measurement

---

## Scoring
- 22-26/26 = Outstanding grabber mail — deploy
- 18-21/26 = Strong concept — refine connection
- 12-17/26 = Needs work — grabber and message disconnected
- Below 12 = Not yet dimensional mail — revisit concept
