# Magnetic Marketing Checklist — 3-Pillar System Check

## Source
Dan Kennedy, "Magnetic Marketing" (2018) + Magnetic Marketing Course

## Purpose
Validate a Magnetic Marketing system against the 3-Pillar framework:
Attract, Convert, Retain. A system is only as strong as its weakest
pillar. All three must function for the system to produce results.

---

## PILLAR 1: ATTRACT (Stop Chasing, Start Attracting)

### Lead Magnets
- [ ] Minimum 3 lead magnets designed for different awareness levels
- [ ] Each lead magnet solves a SPECIFIC problem (not generic "free consultation")
- [ ] Lead magnets have perceived standalone value (worth paying for)
- [ ] Lead magnets are branded and professional
- [ ] Lead magnets create curiosity about paid offerings (without bait-and-switch)
- [ ] Lead magnets are tested and convert at acceptable rate

### USP (Unique Selling Proposition)
- [ ] USP clearly articulated in one sentence
- [ ] USP answers "Why you vs. everyone else, including doing nothing?"
- [ ] USP is used consistently across all marketing materials
- [ ] USP is specific and provable (not vague claims)
- [ ] USP differentiates from ALL competitors

### Attraction Ads/Copy
- [ ] All ads are direct response (not institutional)
- [ ] Ads have clear call to action (request the lead magnet)
- [ ] Ads use Kennedy headline formulas
- [ ] Ads include fascination bullets where appropriate
- [ ] Ads are tracked with unique codes/URLs/phone numbers
- [ ] Multiple ad variations created for testing

### Attraction Channels
- [ ] At least 3 marketing channels active
- [ ] Each channel has tracking/measurement
- [ ] Channel mix includes offline (direct mail) where appropriate
- [ ] Channels reach the target market where they congregate
- [ ] Budget allocated based on ROI, not habit

---

## PILLAR 2: CONVERT (Turn Leads Into Customers)

### Shock-and-Awe Package
- [ ] Package has minimum 5 elements
- [ ] Welcome letter is personalized (not form letter)
- [ ] Authority materials included (book, report, DVD, case studies)
- [ ] Testimonial compilation included
- [ ] FAQ / objection-crushing document included
- [ ] Time-sensitive offer enclosed
- [ ] Premium/widget enclosed
- [ ] Package OVERWHELMS with value and proof
- [ ] Package looks professional and premium

### Multi-Step Follow-Up Sequence
- [ ] Minimum 5 follow-up touches designed
- [ ] Follow-up is multi-MEDIA (mail + email + phone + other)
- [ ] Each touch has a different angle (not just reminders)
- [ ] Timeline mapped: Day 0, 3, 7, 14, 21, 30
- [ ] Urgency escalates with each touch
- [ ] Final touch has "last chance" positioning
- [ ] Non-responder sequence branches off at appropriate point

### Conversion Offer
- [ ] Offer is irresistible (hard to say no)
- [ ] Offer has clear value stacking
- [ ] Guarantee reverses risk
- [ ] Deadline is real and enforced
- [ ] Multiple response channels available (phone, web, mail)
- [ ] Premium/widget incentivizes immediate response

### Conversion Metrics
- [ ] Lead-to-customer conversion rate tracked
- [ ] Cost per acquisition calculated
- [ ] Time-to-conversion measured
- [ ] Follow-up sequence effectiveness tracked by step
- [ ] A/B testing plan in place

---

## PILLAR 3: RETAIN (Keep Customers and Maximize Value)

### Ongoing Communication
- [ ] Monthly newsletter or communication (Kennedy prefers print)
- [ ] Content is valuable, not just promotional
- [ ] Communication reinforces brand and relationship
- [ ] Personal touches included (birthday, anniversary)
- [ ] Communication schedule is consistent (never go dark)

### Customer Ascension
- [ ] Back-end offers designed (upsells, cross-sells)
- [ ] VIP/Inner Circle program available
- [ ] Premium tier with higher price point exists
- [ ] Ascension path is natural and value-based
- [ ] Customer lifetime value tracked and optimized

### Referral System
- [ ] Systematic referral program in place (not hoping)
- [ ] Referral expectations set at onboarding
- [ ] Referral request made at moment of maximum satisfaction
- [ ] Referral tools provided (cards, scripts, kits)
- [ ] Referrals rewarded immediately and tangibly
- [ ] Referrers recognized publicly
- [ ] Endorsed mailing partnerships established

### Reactivation
- [ ] Lost/dormant customers identified
- [ ] Reactivation campaign designed (3-step minimum)
- [ ] Reactivation runs quarterly
- [ ] Special "come back" offer created
- [ ] Win-back results tracked

### Retention Metrics
- [ ] Customer retention rate tracked
- [ ] Customer lifetime value calculated
- [ ] Referral rate measured
- [ ] Churn reasons documented
- [ ] Reactivation success rate tracked

---

## 9 Fatal Marketing Mistakes Cross-Check

Verify NONE of these Fatal Mistakes exist in your system:

- [ ] #1: No institutional/image advertising (all DR)
- [ ] #2: USP is clear and present
- [ ] #3: Target market is specific (not everyone)
- [ ] #4: List building is active and systematic
- [ ] #5: Follow-up system is multi-step and multi-media
- [ ] #6: Multi-step campaigns (not single-touch)
- [ ] #7: Testing is planned and executed
- [ ] #8: ROI tracking on every channel
- [ ] #9: Referral system is systematic (not accidental)
