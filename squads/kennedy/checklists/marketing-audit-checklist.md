# Marketing Audit Checklist — Full Marketing Diagnostic

## Source
Dan Kennedy, compiled from: No B.S. Direct Marketing, Midas Touch Course,
Ultimate Marketing Plan, Magnetic Marketing

## Purpose
Comprehensive marketing audit checklist covering all Kennedy diagnostic
frameworks. Use for quarterly audits, pre-engagement assessments, or
whenever marketing results are underperforming expectations.

---

## SECTION 1: Results Triangle Assessment

### Message
- [ ] Core message is clear and compelling
- [ ] Message addresses a specific pain point or desire
- [ ] Message differentiates from competitors
- [ ] Message is consistent across all channels
- [ ] Message includes a BIG PROMISE backed by proof
- [ ] Message would pass the "cocktail party test" (said aloud, would intrigue?)

### Market
- [ ] Target market is specifically defined (not "everyone")
- [ ] Demographics documented (age, income, location, occupation)
- [ ] Psychographics documented (fears, desires, beliefs, frustrations)
- [ ] Money Pyramid tier identified
- [ ] Customer avatar is based on REAL data (not assumptions)
- [ ] Market size is large enough to sustain the business

### Media
- [ ] Media channels match where target market congregates
- [ ] Multiple media channels in use (not dependent on one)
- [ ] Each channel has tracking/measurement
- [ ] Direct mail included in the mix (Kennedy's preferred)
- [ ] Digital channels used with DR principles (not vanity)
- [ ] Media budget allocated based on ROI data

### Triangle Alignment
- [ ] Message matches what the market wants to hear
- [ ] Media reaches where the market actually is
- [ ] All three elements reinforce each other
- [ ] No single element is significantly weaker than others
