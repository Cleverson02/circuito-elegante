# Results Triangle Checklist

> Agent: kennedy-chief
> Source: No B.S. Direct Marketing — Results Triangle (Message-Market-Media)
> Version: 1.0
> Use: Before any marketing campaign to validate the three pillars are aligned

---

## Pre-Requisites
- [ ] Campaign or project scope defined
- [ ] Budget range established
- [ ] Timeline determined

---

## PILLAR 1: RIGHT Market

### 1.1 Market Identification
- [ ] Target market defined with demographics (age, income, location, occupation)
- [ ] Psychographics mapped (fears, frustrations, desires, aspirations)
- [ ] Buying behavior understood (how they buy, when, triggers)
- [ ] Market size estimated (addressable audience)

### 1.2 Starving Crowd Validation
- [ ] Market has urgent, desperate need (not just mild interest)
- [ ] Market has ability to pay (not just desire)
- [ ] Market is accessible through available media
- [ ] Market is already spending money to solve this problem

### 1.3 Money Pyramid Position
- [ ] Know which segment of the Money Pyramid you are targeting
- [ ] 1% Ultra-Rich / 4% Rich / 15% Comfortable / 60% Paycheck / 20% Broke
- [ ] Marketing approach matches the pyramid segment
- [ ] Price point appropriate for target segment

---

## PILLAR 2: RIGHT Message

### 2.1 Core Message
- [ ] One clear, compelling message (not multiple competing messages)
- [ ] Message addresses the prospect's #1 problem or desire
- [ ] Message is stated in the prospect's language (not industry jargon)
- [ ] Message creates emotional response (not just logical understanding)

### 2.2 USP Integration
- [ ] USP is embedded in the message
- [ ] Message differentiates from all competitors
- [ ] Message answers "Why should I choose YOU?"
- [ ] Message is specific and provable

### 2.3 Proof Elements
- [ ] Message is supported by testimonials
- [ ] Case studies or success stories included
- [ ] Credibility elements present (credentials, media, endorsements)
- [ ] Preponderance of proof (multiple types of proof, not just one)

---

## PILLAR 3: RIGHT Media

### 3.1 Media Selection
- [ ] Media matches where the target market consumes information
- [ ] Multiple channels planned (not single-channel dependency)
- [ ] Media cost-per-acquisition is within acceptable range
- [ ] Media allows for tracking and measurement

### 3.2 A-Pile Compliance (for direct mail)
- [ ] Mail piece would be sorted into the A-Pile (not B-Pile)
- [ ] Envelope looks personal (not mass-mailed)
- [ ] Dimensional or grabber elements considered
- [ ] Real stamps (not metered) if using direct mail

### 3.3 Multi-Channel Strategy
- [ ] At least 2 media channels planned
- [ ] Online and offline channels considered
- [ ] Follow-up sequence uses different media than acquisition
- [ ] Retargeting strategy in place for digital channels

---

## TRIANGLE ALIGNMENT

### The Integration Test
- [ ] Message speaks directly to the chosen Market
- [ ] Media reaches the chosen Market effectively
- [ ] Media format supports the Message (e.g., long copy needs right channel)
- [ ] All three pillars reinforce each other (no contradiction)
- [ ] If any one pillar changes, the other two are adjusted

### The Weak Link Test
- [ ] Identified which pillar is currently weakest
- [ ] Resources allocated to strengthen the weak pillar
- [ ] Not over-investing in strong pillars while weak pillar drags results

---

## Validation
- [ ] Results Triangle diagram drawn with specific content in each corner
- [ ] Team alignment on all three pillars (no conflicting interpretations)
- [ ] Budget allocated proportionally across all three pillars
- [ ] Measurement plan in place to track each pillar's contribution

---

## Scoring
- 28-32/32 = Triangle is aligned — launch campaign
- 22-27/32 = Minor gaps — fix before launch
- 16-21/32 = Significant gaps — do NOT launch until fixed
- Below 16 = Triangle is broken — full redesign needed
