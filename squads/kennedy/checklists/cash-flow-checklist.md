# Five Keys to Multiplied Cash Flow Checklist

> Agent: kennedy-business
> Source: The Ultimate No B.S. Business Success Book — Cash Flow Multiplication
> Version: 1.0
> Use: When optimizing business cash flow using Kennedy's 5 multiplication keys

---

## Pre-Requisites
- [ ] Current revenue and expense data available
- [ ] Customer acquisition cost (CAC) known
- [ ] Customer lifetime value (LTV) calculated or estimated
- [ ] Transaction history available

---

## KEY 1: Increase Number of Customers
- [ ] Lead generation system active and producing
- [ ] Multiple lead sources in use (not single-channel)
- [ ] Cost per lead tracked by source
- [ ] Referral system generating new customers
- [ ] Lost customer reactivation campaign running
- [ ] Target: ___% increase in customer count

## KEY 2: Increase Average Transaction Value
- [ ] Upsell offers presented at point of sale
- [ ] Cross-sell recommendations systematized
- [ ] Premium tier or VIP option available
- [ ] Bundling increases the average ticket
- [ ] Price increases evaluated (when was the last one?)
- [ ] Target: ___% increase in average transaction

## KEY 3: Increase Frequency of Purchase
- [ ] Follow-up system encourages repeat purchases
- [ ] Monthly newsletter or regular communication in place
- [ ] Loyalty program or membership incentivizes returns
- [ ] New offers presented regularly to existing customers
- [ ] Seasonal or event-based promotions planned
- [ ] Target: ___% increase in purchase frequency

## KEY 4: Increase Retention / Reduce Attrition
- [ ] Customer satisfaction monitored systematically
- [ ] At-risk customers identified before they leave
- [ ] Win-back campaigns for recently lost customers
- [ ] Customer appreciation program active
- [ ] Exit interviews conducted for departing customers
- [ ] Target: ___% reduction in attrition

## KEY 5: Increase Profit Margins
- [ ] Pricing reviewed against market and value delivered
- [ ] Costs reduced without sacrificing quality
- [ ] Low-profit products/services evaluated (keep or kill?)
- [ ] Operational efficiency improved
- [ ] Premium positioning allows higher margins
- [ ] Target: ___% increase in profit margins

---

## THE MULTIPLICATION EFFECT

Kennedy's key insight: Even SMALL improvements in each key multiply together.

Example:
- 10% more customers x 10% higher transaction x 10% more frequent = 33% revenue increase
- All 5 keys improved 10% each = 61% increase in cash flow

### Calculate Your Multiplication
- Current customers: ___
- Current avg transaction: $___
- Current frequency: ___ per year
- Current retention rate: ___%
- Current margin: ___%

### After Improvement (even 10% each)
- New customers: ___
- New avg transaction: $___
- New frequency: ___ per year
- New retention: ___%
- New margin: ___%
- **Projected revenue increase: ___%**

---

## Validation
- [ ] All 5 keys assessed with current metrics
- [ ] Improvement targets set for each key
- [ ] Action plan for each key documented
- [ ] Multiplication effect calculated
- [ ] Monthly review scheduled

---

## Scoring
- 26-30/30 = All 5 keys actively optimized — monitor and refine
- 20-25/30 = Strong foundation — activate remaining keys
- 14-19/30 = Significant opportunity — major keys untouched
- Below 14 = Cash flow not being managed — start with Key 1
