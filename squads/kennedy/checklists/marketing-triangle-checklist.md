# Marketing Triangle Audit Checklist

> Agent: kennedy-audit
> Source: Magnetic Marketing + Ultimate Marketing Plan — Marketing Triangle
> Version: 1.0
> Use: When auditing overall marketing effectiveness using the three-legged model

---

## Pre-Requisites
- [ ] Business financial data available
- [ ] Marketing campaigns and materials gathered
- [ ] Customer data accessible
- [ ] Willingness to be honest about weaknesses

---

## LEG 1: MESSAGE — What You Say

### 1.1 Core Message
- [ ] Single, clear core message identified
- [ ] Message addresses prospect's #1 pain or desire
- [ ] Message is stated in prospect's language (not yours)
- [ ] Message differentiates from every competitor
- [ ] USP is embedded in the core message

### 1.2 Message Consistency
- [ ] Same core message across all channels
- [ ] Message is not diluted with multiple competing messages
- [ ] Every team member can articulate the message
- [ ] Message has been tested with actual prospects

### 1.3 Message Power
- [ ] Message creates emotional reaction
- [ ] Message compels action (not just awareness)
- [ ] Message passes the "so what?" test
- [ ] Message would interrupt a distracted prospect

---

## LEG 2: MARKET — Who You Say It To

### 2.1 Market Definition
- [ ] Target market defined beyond basic demographics
- [ ] Psychographic profile complete (fears, desires, beliefs)
- [ ] Buying behavior mapped (how, when, why they buy)
- [ ] Market is reachable through available media

### 2.2 Market Selection
- [ ] Market has urgent need (starving crowd)
- [ ] Market has ability and willingness to pay
- [ ] Market is large enough to sustain business goals
- [ ] Market can be reached cost-effectively

### 2.3 Market Knowledge
- [ ] Know what keeps them awake at 3 AM
- [ ] Know their biggest frustrations and desires
- [ ] Know who they trust and follow
- [ ] Know where they spend time and attention

---

## LEG 3: MEDIA — How You Deliver the Message

### 3.1 Media Selection
- [ ] Media channels match where target market is
- [ ] Multiple channels used (not single-channel dependency)
- [ ] Each channel has DR elements (offer, deadline, CTA, tracking)
- [ ] Cost per acquisition known for each channel

### 3.2 Media Execution
- [ ] Direct mail included in the mix (Kennedy's preference)
- [ ] Online and offline channels integrated
- [ ] Follow-up sequences cross multiple media
- [ ] Media budget allocated based on ROI, not habit

### 3.3 Media Measurement
- [ ] Every channel tracked with unique identifiers
- [ ] ROI calculated per channel
- [ ] Underperforming channels identified
- [ ] Winners scaled, losers cut

---

## TRIANGLE BALANCE

### Alignment Assessment
- [ ] Message speaks to the chosen Market
- [ ] Media reaches the chosen Market
- [ ] Media format supports the Message length/complexity
- [ ] All three legs are equally strong (no one-legged stool)

### Weakest Leg Identification
- [ ] Weakest leg identified: MESSAGE / MARKET / MEDIA
- [ ] Resources redirected to strengthen weakest leg
- [ ] Timeline for bringing weakest leg to parity
- [ ] Not over-investing in strongest leg (diminishing returns)

---

## Validation
- [ ] All three legs audited with specific findings
- [ ] Weakest leg identified and action plan created
- [ ] Triangle alignment verified (no contradictions)
- [ ] Measurement systems in place for ongoing monitoring
- [ ] Quarterly Triangle audit scheduled

---

## Scoring
- 28-32/32 = Balanced, powerful Marketing Triangle
- 22-27/32 = Strong triangle — one leg needs strengthening
- 16-21/32 = Imbalanced — significant weakness in one or more legs
- Below 16 = Triangle is broken — fundamental redesign needed
