# Direct Mail Piece Checklist

> Agent: kennedy-media
> Source: No B.S. Direct Marketing + Ultimate Sales Letter — Direct Mail Mastery
> Version: 1.0
> Use: When creating any direct mail piece (letter, postcard, package, dimensional)

---

## Pre-Requisites
- [ ] Target recipient list quality verified
- [ ] Mailing format selected (letter, postcard, self-mailer, dimensional)
- [ ] Budget per piece calculated
- [ ] Tracking mechanism ready (unique phone, URL, code)

---

## STEP 1: The Envelope/Outer

### 1.1 A-Pile Compliance
- [ ] Looks personal, not mass-mailed
- [ ] Hand-addressed or simulated hand-addressing
- [ ] Real stamps (not metered or bulk rate indicia)
- [ ] No window envelope (unless bill simulation is intentional)
- [ ] Teaser copy (if used) creates irresistible curiosity

### 1.2 Format Selection
- [ ] Size is non-standard (stands out from regular mail)
- [ ] Color is notable (not standard white #10)
- [ ] Texture or weight is different
- [ ] Dimensional/lumpy elements included if applicable

---

## STEP 2: The Letter Content

### 2.1 Opening
- [ ] Personal salutation (Dear [Name], not Dear Friend)
- [ ] First paragraph hooks immediately (1-3 sentences)
- [ ] Opens with prospect's problem or desire (not your product)
- [ ] Pest-to-Welcome-Guest technique applied

### 2.2 Body
- [ ] Story included (personal, client, or case study)
- [ ] Benefits prioritized over features
- [ ] Fascination bullets create curiosity
- [ ] Proof elements throughout (testimonials, stats, case studies)
- [ ] Problem agitated before solution presented

### 2.3 Offer
- [ ] Clear, specific offer stated
- [ ] Value anchored (total value vs. price)
- [ ] Guarantee prominently displayed
- [ ] Bonuses listed with individual values
- [ ] Widget/premium included if applicable

### 2.4 Close
- [ ] Urgency — specific deadline stated
- [ ] Scarcity — limited quantity or availability
- [ ] Clear call to action (exact instructions)
- [ ] Multiple response channels (phone, URL, reply mail)
- [ ] P.S. lines (minimum 2, each could sell on its own)

---

## STEP 3: Design and Production

### 3.1 Typography
- [ ] Easy to read font (serif for letters, sans-serif for headlines)
- [ ] Key phrases bolded, underlined, or highlighted
- [ ] Short paragraphs (2-4 lines)
- [ ] Subheadings every 3-5 paragraphs
- [ ] Wide margins (not wall-of-text)

### 3.2 Visual Elements
- [ ] Photos with captions (people read captions)
- [ ] Testimonial callout boxes
- [ ] Guarantee certificate box
- [ ] Order form highlighted
- [ ] Arrows, check marks, or annotations guide the eye

---

## STEP 4: Tracking and Testing

### 4.1 Tracking
- [ ] Unique tracking code on the piece
- [ ] Dedicated phone number or extension
- [ ] Dedicated URL or landing page
- [ ] Source code on reply card or order form

### 4.2 Testing Plan
- [ ] Split test designed (A/B on one variable)
- [ ] Control piece identified (the "to beat")
- [ ] Sample size sufficient for statistical significance
- [ ] Test results timeline established

---

## Validation
- [ ] Piece passes A-Pile sort test (5 people, 4+ would open)
- [ ] Read aloud sounds like one person talking to another
- [ ] All tracking mechanisms verified and working
- [ ] Offer is irresistible and clear
- [ ] No spelling or pricing errors

---

## Scoring
- 34-38/38 = Production-ready direct mail piece
- 28-33/38 = Strong piece — minor refinements
- 20-27/38 = Needs work — several DR elements missing
- Below 20 = Institutional mail masquerading as DR — rewrite
