## SECTION 3: POST-WRITING

### Step 23: Self-Edit Checklist
- [ ] Read aloud — does it flow conversationally?
- [ ] Is the headline the strongest possible?
- [ ] Does every paragraph earn the next?
- [ ] Are there enough fascination bullets?
- [ ] Is the offer irresistible?
- [ ] Is the guarantee bold and prominent?
- [ ] Is the CTA crystal clear?
- [ ] Are the P.S. sections strong?

### Step 24: DR Compliance Check
- [ ] Specific offer present
- [ ] Deadline/urgency present
- [ ] Clear CTA present
- [ ] Tracking mechanism present
- [ ] Multiple response channels available

### Step 25: Proof Audit
- [ ] Minimum 4 proof types used
- [ ] Testimonials have full names and locations
- [ ] Claims supported by evidence
- [ ] Credentials stated
- [ ] Third-party validation included

### Step 26: A-Pile/B-Pile Check (if mailed)
- [ ] Envelope passes A-Pile test
- [ ] B-Pile death markers: 0
- [ ] First 10 seconds grab attention

### Step 27: 10 No B.S. Rules Check
- [ ] Rule 1: Offer present
- [ ] Rule 2: Reason to respond now
- [ ] Rule 3: Clear instructions
- [ ] Rule 4: Tracking in place
- [ ] Rule 5: Brand-building is no-cost byproduct
- [ ] Rule 6: Follow-up planned
- [ ] Rule 7: Strong copy
- [ ] Rule 8: Looks like DR (not corporate)
- [ ] Rule 9: Results will be measured
- [ ] Rule 10: Disciplined execution

### Step 28: Final Sign-Off
- [ ] Would I buy this? (honest answer)
- [ ] Does this pass the "kitchen table" test? (reads like a friend talking)
- [ ] Is the strongest benefit in the headline?
- [ ] Am I proud of this letter?
