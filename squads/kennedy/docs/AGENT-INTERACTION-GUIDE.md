## Troubleshooting Agent Interactions

**Problem:** Agent says "this isn't my discipline"
**Solution:** You routed to the wrong agent. Check interaction guide and try primary agent.

**Problem:** Two agents give contradicting advice
**Solution:** They're optimizing for different goals. Load Kennedy-Chief to arbitrate.

**Problem:** Agent won't proceed without more information
**Solution:** This is correct. Provide the requested information. Garbage in = garbage out.

**Problem:** Output doesn't match your expectation
**Solution:** Give clearer brief or more context. Agents need specific inputs to generate specific outputs.

**Problem:** You need multiple agents but unclear on sequence
**Solution:** Load Kennedy-Chief first. Chief will route you in optimal sequence.

## Agent Etiquette

1. **Be Specific:** Vague requests get vague outputs
2. **Provide Context:** More context = better recommendations
3. **Give Feedback:** Tell agents what worked/didn't
4. **Follow Handoffs:** When agent hands off, load the receiving agent
5. **Trust the System:** Agents are trained on Kennedy's decades of work
